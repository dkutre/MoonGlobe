Прорисовка текстурированной сферы с использованием OpenGL4+, Magnum.

Установка:

    git clone https://github.com/dkutre/MoonGlobe.git

Зависимости: требуется установленная libJPEG.

Сборка:

    ./rebuild.sh
    ./build.sh


Запуск:

    ./bin/Releaser/m2-sphere

Используется:

    Magnum, libJPEG.
